Gerenciamento de Compras

JavaScript
JS/compras.js

1.LocalStorage
salvarLocalStorage() e carregarLocalStorage() mantêm os dados persistidos no navegador.

2.Renderizar a tabela
Função renderizarTabela limpa e monta a tabela conforme o array (ou filtro).
Se estiver no modo seleção, mostra checkboxes para cada item.
Cada checkbox controla o set selecionados.
Botões individuais (editar e excluir) são criados para cada linha.
Atualiza visibilidade do cabeçalho da coluna de seleção.

3.Botões Editar e Excluir múltiplos
Visibilidade controlada por quantos itens estão selecionados.
Editar só aparece se 1 item selecionado.
Excluir aparece se >=1 selecionados.

4. Adicionar, excluir e editar produto(s)

5.Toggle de seleção
Botão “Selecionar” ativa ou desativa a seleção (checkboxes).
Desativa selecionados e atualiza tabela.

6.Filtro em tempo real
Filtra array por nome do produto conforme o que o usuário digita.
Renderiza só os filtrados.

JS/storage.js

1.Quando adicionar uma compra:
O formulário chama uma função no compras.js que pega os dados do input.
Essa função atualiza o array de compras e chama salvarCompras(compras) para gravar no localStorage.

2.Quando a página carrega:
O compras.js chama carregarCompras() para montar a tabela com os dados salvos.

3.Editar e excluir:
Também usam carregarCompras() para ler e salvarCompras() para reescrever os dados atualizados.
